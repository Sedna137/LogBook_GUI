# LogBook_GUI
A Python GUI for easy logging. Such as lab notes, diary, and reports.

Author Jeena A. V., created in 2019, Updated on 11.03.2026

written in Python 3.14.2
- Requirements: tkinter

Save this Python file in the desired directory e.g., 'LogbookDir'
- open conda terminal
- cd D:\LogbookDir
- python LogDiary.py

This should launch a user interface as shown in 'LogDiary_Fig1.png'. Entries can be given and click either 'save all' or 'save notes'. It will then save entries to the desired directory.
One can change the filename and directory at 'SaveLogs(self)' function body.
